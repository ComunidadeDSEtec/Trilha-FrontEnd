# Trilha-FrontEnd
Um dos maiores desafios na área de TI é saber por onde começar, em uma área tão vasta se perder chega a ser um evento canônico. Eu mesmo já enfrentei esse problema, com isso em mente criei esse guia para todos que querem aprender desenvolvimento frontend.

## Recomendações adicionais
<ul>
  <li>HTML, linguagem de marcação. Procure exercitar principalmente o uso de tags semânticas, "Header", "Nav" e não só "div";</li>
  <li>CSS, linguagem de estilização. É impossível aprender TODAS as tags CSS, então foque em construir uma base sólidad. Ademais aprenda, desing responsivo, media queries, display grid e flex, finalizando com @keyframe;</li>
  <li>Frameworks CSS, biblioteca de código, aprenda a utilizar as bibliotecas com projetos práticos utilizando todos os conhecimentos desenvolvidos;</li>
  <li>Lógica de programação em javaScript, aprenda os comandos básicos e a lógica, diretamente na linguagem que você irá utilizar.
  O que procurar aprender em JS passo a passo:
    <ol>
      <li>DOM;</li>
      <li>Funções;</li>
      <li>Modularização;</li>
      <li>Programação Orientada a Objetos (POO);</li>
      <li>Expressões regulares (RegEX);</li>
      <li>Json;</li>
      <li>Ajax;</li>
      <li>Node.JS.</li>
    </ol>
  </li>
  <li>Frameworks JS, biblioteca de código, aprenda a utilizar as bibliotecas com projetos práticos utilizando todos os conhecimentos desenvolvidos;</li>
  <li>Vesionamento e hospedagem de código, aprenda a utilizar o Git, assim você conseguirá se manter organizado, e por fim GitHub para hospedar seus projetos/sites em algum lugar de forma gratuita;</li>
  <li>User Experience (UX) e User Interface (UI), aprender a criar projetos chaamtivos, responsivos e que possuam um layout condizente com o projeto;</li>
  <li>Projetos práticos, agora chegou a hora de desenvolver projetos e bombardear seu perfil do GitHub, esse é um ótimo método para divulgar suas habilidades e entrar com mais facilidade no mercado de trabalho.</li>
</ul>
